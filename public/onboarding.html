Onboarding — Current vs Future State

The gap between manual and automated onboarding is not about technology. It is about how much of your team's time you are willing to spend on repetitive work.

Side-by-Side Comparison

MilestoneCurrent State (Manual)Future State (Automated + AI)
Day 0 — SignCSM manually sends welcome email + Calendly link. Customer waits 1–2 days for first contact.Instant: automated welcome email, in-app onboarding checklist appears on first login, setup wizard begins.
Day 1 — First LoginCSM checks if customer logged in. Sends follow-up if not. Manual note in CRM.In-app prompt: "Let us get you set up — this takes 8 minutes." AI assistant available immediately. Login event triggers CRM update.
Day 3 — SetupKickoff call scheduled at sign. CSM walks through setup live on screen share.If setup checklist under 30% complete: automated nudge email. "You have not connected your data source yet — here is a 2-min video."
Day 7 — First ValueCSM reviews usage data manually. Calls if no usage. Emails if looks OK.Health score calculated automatically. If no first value milestone hit: triggered CSM task + automated email from CSM.
Day 14 — Check-inCSM schedules and runs 14-day call. Takes notes. Updates CRM manually.For SMB: automated digest email with usage summary. For MM/Enterprise: CSM auto-briefed with usage report before call.
Day 30 — ReviewCSM prepares review manually. Pulls usage, builds slide, schedules call.Automated 30-day report emailed to customer. CSM gets pre-built deck with live data. Meeting scheduled automatically if health below 60.
Day 90 — SuccessCSM preps success review, identifies expansion opportunities manually.AI surfaces expansion signals. CSM gets prompt: "Acme Corp looks ready for seats upgrade — here is the data."

Current State Timeline

Manual touchpoints only. Every step requires a human decision.

D0
Day 0 — Contract signed

CSM notified via Slack. Manual welcome email sent within 24h. Calendly link shared.

D3
Day 3 — Kickoff call

60-min call. CSM shares Google Doc checklist. Homework assigned to customer.

D14
Day 14 — Manual follow-up

CSM checks if checklist was completed. Email if not. No automated fallback.

D30
Day 30 — Check-in call

CSM manually preps, schedules, runs. Updates notes in CRM. No automation.

D90
Day 90 — Success review

If CSM remembered to book it. ROI data pulled manually. Expansion pitched if gut says yes.

Key Takeaway

The goal of future-state onboarding is not to remove the human — it is to make sure the human shows up at the right moment, with the right context, for the right customer. Automate the predictable. Reserve the CSM for the unpredictable.
